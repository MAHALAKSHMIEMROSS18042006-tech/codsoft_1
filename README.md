# 🎮 Tic-Tac-Toe AI using Minimax Algorithm

## 📌 Overview

This project is a command-line Tic-Tac-Toe game where a human player competes against an AI agent. The AI uses the Minimax algorithm to make optimal decisions, ensuring it never loses.

---

## 🧠 Concept Used

* Minimax Algorithm (Game Theory)
* Recursive Decision Making
* State Space Search

---

## ⚙️ Features

* Unbeatable AI opponent
* Human vs Computer gameplay
* Detects win, loss, and draw conditions
* Clean and modular Python implementation

---

## 🛠️ Technologies Used

* Python

---

## 🚀 How It Works

The AI evaluates all possible future moves using the Minimax algorithm:

* Maximizes its own chances of winning
* Minimizes the opponent’s chances
* Assigns scores to outcomes:

  * +1 → AI wins
  * 0 → Draw
  * -1 → Human wins

Based on this, the AI selects the optimal move.
